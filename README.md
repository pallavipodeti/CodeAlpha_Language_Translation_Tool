# 🌍 Language Translation Tool

## 📌 Project Overview

This project is a Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text between multiple languages using Google Translation API. It also includes Copy Translation and Text-to-Speech features.

## 🚀 Features

- Real-time language translation
- Multiple language support
- User-friendly Streamlit interface
- Copy translated text
- Text-to-Speech functionality

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- gTTS
- Pyperclip

## ⚙️ Installation

1. Install required packages

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
streamlit run app.py
```
